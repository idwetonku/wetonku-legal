Privacy Policy

Tanggal efektif:

Wetonku menghormati privasi Anda. Aplikasi ini dirancang untuk berfungsi secara offline dan tidak mewajibkan pembuatan akun.

1) Data yang kami kumpulkan

Kami tidak mengumpulkan data pribadi (misalnya nama, email, nomor telepon), dan tidak memiliki fitur login.

2) Data yang disimpan di perangkat

Aplikasi dapat menyimpan preferensi atau riwayat sederhana secara lokal di perangkat Anda (jika fitur itu diaktifkan). Data lokal ini tidak dikirim ke server mana pun.

3) Pihak ketiga

Wetonku tidak membagikan data Anda ke pihak ketiga. Aplikasi ini tidak menggunakan layanan analitik pihak ketiga untuk melacak aktivitas pengguna (untuk versi offline ini).

4) Izin perangkat

Wetonku tidak memerlukan izin sensitif. Jika di masa depan ada fitur yang memerlukan izin tertentu, kami akan menjelaskan tujuannya secara jelas di aplikasi dan memperbarui kebijakan ini.

5) Keamanan

Karena tidak ada pengiriman data ke server, risiko transmisi data sangat minim. Tetap jaga keamanan perangkat Anda (PIN/biometrik) untuk melindungi data lokal.

6) Perubahan kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal efektif terbaru.

7) Kontak

Jika ada pertanyaan terkait privasi, silakan hubungi kami melalui email pada bagian Kontak.

Catatan: Jika suatu saat Anda menambahkan iklan/analytics/login, bagian ini harus diperbarui sesuai implementasi.
